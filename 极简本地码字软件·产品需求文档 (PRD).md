
### 一、 产品定位与技术基建

* **核心理念：** 极简、高颜值、纯粹本地化、无干扰的沉浸式中文写作体验。
* **目标平台：** Windows 桌面端。
* **技术栈选型：**
* **外壳与底层：** **Tauri 2.0 (Rust)**（负责极速启动、轻量化、本地文件 I/O、ZIP 压缩打包）。
* **前端视图层：** **Vue 3 + TypeScript**（负责界面渲染、状态管理、侧边栏交互）。
* **富文本内核：** **Tiptap**（无头富文本框架，负责“所见即所得”的 Markdown 序列化与排版渲染）。



---

### 二、 数据存储与目录规范

* **数据隔离原则：** 数据完全暴露且存储于本地系统，不强制要求云同步。
* **文件格式：** 正文采用纯 **Markdown (`.md`)** 格式，保证数据的永久可读性。
* **目录结构与排序逻辑（方案 B：索引文件法）：**
* 在 Windows 资源管理器中，文件按默认名称显示（如 `引子.md`, `第一章.md`）。
* 在每个书籍的根目录下生成一个 `book.json` 文件。
* 软件内部的目录树、拖拽排序、以及最终的导出拼接，**全部以 `book.json` 中的记录为准**，与系统的字母排序解耦。



---

### 三、 核心编辑器与排版美学

* **编辑模式：** 所见即所得 (WYSIWYG)，隐藏复杂的 Markdown 语法符号。
* **中文排版专项强化：**
* **自动缩进：** 通过 CSS `text-indent: 2em;` 实现段首自动缩进两字符，**不**在 Markdown 源码中插入物理空格。
* **精细调节：** 开放对字体大小、字体样式（宋体、黑体等本地字体）、行距 (`line-height`)、字间距 (`letter-spacing`)、左右边距的动态调节。


* **沉浸式 UI 与主题：**
* 无多余边框和阴影的极简界面。
* 支持页面背景色切换（主题 Theme 控制）。
* **特色功能：** 支持动态生成的“稿纸网格线”，并要求网格线的高度与文字行高完美对齐，随字体大小自适应缩放。



---

### 四、 批注与灵感系统（特色核心）

* **呈现形式（方案 A：侧边栏联动卡片）：**
* 编辑器内被选中的文字会带有特殊的高亮或下划线。
* 点击或悬停时，界面右侧边栏自动滑出或定位到对应的灵感卡片。


* **底层存储方案（无损隐式存储）：**
* 继续使用单文件 Markdown 存储。
* 灵感内容在底层被转化为兼容的 HTML 注释标签：``。
* 保证用第三方阅读器打开 `.md` 文件时，正文依旧纯净，灵感内容被自动隐藏。



---

### 五、 内容组织与界面交互

* **全局首页工作台：**
* 卡片式/列表式展示，可以创建或选择“小说”或“随笔集”。
* 支持为每本书/集子设置和展示“简介”或“大纲”。


* **多级目录管理（小说模式）：**
* 支持新增卷、新增章节、重命名。
* 支持在左侧边栏通过拖拽直接调整卷和章的先后顺序（实时回写 `book.json`）。


* **时间线管理（随笔模式）：**
* 记录随笔的创建时间、最后修改时间以及**地点**信息（通过 YAML Frontmatter 或独立 JSON 记录）。
* 以垂直时间轴（Timeline）UI 进行升序/降序展示。


* **状态栏：** 实时字数统计，自动记录单次/总计写作时长。

---

### 六、 版本控制与安全灾备

* **历史版本管理（方案 A：隐藏文件夹）：**
* 在每一本书的根目录下，自动创建 `.history` 隐藏文件夹。
* 每次触发保存时，除更新原文件外，自动在 `.history` 中拷贝一份以 `章节名_时间戳.md` 命名的历史快照。


* **一键全量备份与恢复：**
* 调用 Rust 底层能力，将当前小说整个文件夹（包含 `.md` 正文、`book.json` 索引、`.history` 历史记录）打包为单一的 `.zip` 压缩包导出。
* 支持从 `.zip` 文件直接解析覆盖，完成数据恢复。



---

### 七、 最终作品导出系统

* **TXT 纯文本导出（净化处理）：**
* 依据 `book.json` 的顺序自动合并多章节。
* 自动运行正则替换，**彻底剔除**所有的灵感注释 `` 以及多余的 Markdown 标记符号。


* **PDF 电子书导出（出版级排版）：**
* 不依赖后端繁重的 PDF 生成库，利用浏览器的原生 CSS Paged Media `@page` 规则渲染。
* 完美复刻软件内的中文排版（缩进、字体、边距）。
* 实现章节自动换页强制分页、页眉生成书名、页脚生成页码。
