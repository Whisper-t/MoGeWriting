这是一份为你量身定制的完整项目实施计划。采用 Tauri 2.0 (Rust) + Vue 3 + Tiptap 的架构，彻底贯彻极简、本地化、纯粹写作的理念。采用本地纯 Markdown 文件结构，能够保证数据的绝对掌握与透明度，这与使用 Obsidian 等工具进行知识管理时的控制感是一脉相承的。

### 总体架构与目录规范

* **前端视图：** Vue 3 (处理界面交互、拖拽、侧边栏)
* **编辑器内核：** Tiptap (无头富文本框架，完美序列化 Markdown 与 HTML 注释)
* **底层与跨平台：** Tauri 2.0 (Rust，负责文件读写、隐藏文件夹、本地打包导出)
* **存储结构：** 本地 Markdown 文件 + `book.json` 索引文件 + `.history` 隐藏文件夹

---

### 阶段一：底层架构与文件管理引擎 (第 1-2 周)

本阶段主要在 Rust 端与操作系统的文件系统进行交互，搭建稳固的本地数据基座。

* **项目初始化：** 使用 Tauri CLI 快速生成 Tauri 2.0 + Vue 3 + Vite 的项目骨架，并清理多余的样板代码。
* **工作区 API 开发：** 在 Rust 端编写接口，实现本地文件夹的创建、读取、重命名，以及 `.md` 文件的无损读写。
* **索引驱动加载：** 编写 `book.json` 解析逻辑，使前端目录树强依赖此 JSON 文件进行排序，无视 Windows 默认的文件名排序规则。
* **版本快照引擎：** 拦截编辑器的保存动作，在写入源文件的同时，于同级 `.history` 隐藏文件夹中静默拷贝一份带有时间戳的 `.md` 副本。

---

### 阶段二：所见即所得编辑器与中文排版 (第 3-4 周)

本阶段是软件的灵魂，重点打造无干扰且符合中文直觉的纯净写作体验。

* **富文本内核接入：** 引入 Tiptap 框架，剥离默认 UI，仅保留加粗、斜体、各级标题等核心 Markdown 节点的解析与渲染。
* **中文排版专项优化：** 编写全局 CSS，强制应用 `text-indent: 2em;` 实现段前自动缩进，并开放 `line-height` (行距) 和 `letter-spacing` (字间距) 的动态调节接口。
* **沉浸式主题系统：** 利用 CSS 变量构建全局主题控制器，实现背景颜色无缝切换。
* **动态网格线绘制：** 利用 CSS 的 `linear-gradient` 属性在编辑器底层绘制“稿纸网格线”，并监听字体大小变化，通过计算保持网格线与文字行高的完美像素级对齐。

---

### 阶段三：侧边栏批注与灵感系统 (第 5-6 周)

实现你最核心的创新需求，让灵感与正文解耦但又紧密联动。

* **自定义 Tiptap 标记：** 开发一个自定义扩展，允许选中特定文字并绑定一个生成的唯一 UUID，在编辑器内呈现专属的高亮下划线。
* **隐式存储转换：** 修改序列化规则，确保带有 UUID 的文字在保存为 `.md` 时，自动在对应位置生成 `` 的原生 HTML 注释。
* **侧边栏联动渲染：** 在 Vue 前端开发独立侧边栏组件，当光标悬停或点击高亮文字时，侧边栏自动滚动并展开对应的灵感卡片。

---

### 阶段四：界面组装与数据绑定 (第 7-8 周)

将零散的功能模块拼装为高颜值的现代化极简桌面软件。

* **全局工作台开发：** 构建卡片式的首页，用于展示、创建、选择不同的小说卷宗或随笔集，并支持输入和展示书籍简介。
* **随笔时间线视图：** 解析随笔文件夹下所有文件的元数据（创建时间、地点），利用 Vue 渲染出一条按时间降序排列的美观时间轴。
* **目录树与拖拽排序：** 在小说编辑界面的左侧边栏引入拖拽库，实现卷与章节的拖拽排序，并在拖拽结束时实时将新顺序回写更新至本地的 `book.json` 文件。

---

### 阶段五：导出引擎、备份恢复与封版 (第 9-10 周)

完成作品的最后输出通路与整机安全保障，确保软件达到最终交付标准。

* **纯净 TXT 编译：** 编写正则表达式处理器，在导出 TXT 时依据 `book.json` 依次合并章节，彻底清洗掉所有 `` 标签，并将 Markdown 符号转化为纯文本格式。
* **出版级 PDF 渲染：** 利用 Web 标准的 `@page` CSS 规则定义书籍尺寸与页边距，实现章节标题自动强制分页，并调用 Tauri 的 Webview 打印接口一键输出精美 PDF。
* **ZIP 灾备与恢复：** 在 Rust 端集成 ZIP 压缩库，将当前小说目录一键打包导出，同时提供逆向的解压覆盖恢复功能。
* **全局 UI 降噪：** 走查所有界面，剔除多余的边框、生硬的阴影和复杂的交互动画，严格坚守简洁明了的设计准则。
